## ⟡ Currently Engineering

- 🤖 **Enterprise multi-agent platforms** — LangGraph state machines with conditional query routing, semantic retrieval, and source-grounded loops that drive hallucinations toward zero.
- 📚 **Multi-format ingestion engines** — 8 specialized parsers + an adaptive chunker, offloading heavy parsing to distributed **Celery Chords** so nothing times out.
- 🔌 **MCP tool orchestration** — letting LLMs safely read, search, and map external ecosystems (Drive, Notion) through the Model Context Protocol.
- 🔭 **LLMOps observability** — LangSmith trace-linking, token/cost auditing, and high-throughput **FastAPI SSE** streaming for live agent traffic.
